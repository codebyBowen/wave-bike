# 🏍️ Wave Rider - 赛博朋克摩托特技游戏

一个用 Godot 4.5 制作的像素风赛博朋克摩托跑酷游戏。

## 🎮 游戏玩法

- **目标**: 在无尽的波浪地形上骑行，完成空翻特技获得高分
- **控制**:
  - `A` 键 - 向后倾斜（后空翻）
  - `D` 键 - 向前倾斜（前空翻）
  - `ESC` 键 - 暂停游戏
  - `R` 键 - 重新开始（游戏结束后）

## 🎯 得分系统

### 空翻分数
- 1 翻 = 200 分
- 2 翻 = 500 分
- 3 翻 = 1500 分
- 4 翻 = 5000 分
- 5 翻及以上 = 翻数 × 2000 分

### 落地质量倍率
- **PERFECT** (0-5°): 3x 倍率 ⭐
- **GREAT** (5-15°): 2x 倍率
- **GOOD** (15-30°): 1x 倍率
- **>30°**: 摔车 💥

### 连击系统
- 连续成功特技增加连击数
- 连击倍率：
  - 3-4 连击: 2x
  - 5-6 连击: 3x
  - 7-9 连击: 4x
  - 10+ 连击: 5x
- 摔车会打断连击

### 滞空奖励
- 每秒滞空时间 = +50 分

## 🚀 开始使用

### 方式 1：在 Godot 编辑器中运行
1. 打开 Godot 4.5
2. 导入项目：选择 `project.godot` 文件
3. 按 `F5` 或点击"运行项目"

### 方式 2：命令行运行
```bash
godot --path /Users/user/Desktop/projects/wave-bike
```

## 🔧 调试和测试

### 查看控制台输出
游戏会输出详细日志：
- 起飞和着陆事件
- 空翻角度和质量
- 得分计算详情

### 调整参数
在 Godot 编辑器中可以调整：
- `Bike.tscn`: 加速度、最大速度、旋转力矩
- `Terrain.tscn`: 段长度、难度、生成距离
- `Global.gd`: 角度判定范围、得分倍率

## 🐛 已知问题

1. **偶尔卡在地形中**: 可能需要调整碰撞形状
2. **高速时摄像机抖动**: 可以增加 Camera2D 的平滑速度
3. **UI 文字可能在某些分辨率下不居中**: 需要调整锚点

## 📝 技术细节

### 物理参数
- 重力倍率: 1.0
- 摩托车质量: 50kg
- 线性阻尼: 0.1
- 角阻尼: 0.5

### 地形生成
- 每段长度: 400 像素
- 分辨率: 20 点/段
- 提前生成距离: 2000 像素
- 地形类型: 平地、正弦波、颠簸、跳台、山丘

### 空翻检测算法
```gdscript
# 累积旋转角度（度数）
Global.total_rotation += angle_change_per_frame

# 着陆时计算空翻数
var flips = int(abs(Global.total_rotation) / 360.0)
```
